# Rspack project

## Setup

Install the dependencies:

```bash
yarn install
```

## Get started

Start the dev server, and the app will be available at <http://localhost:8080>.

```bash
yarn run dev
```

Build the app for production:

```bash
yarn run build
```

Preview the production build locally:

```bash
yarn run preview
```
